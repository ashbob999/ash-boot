### Features

##### Current Features
- functions
- simple types (int, float, bool, char)
- binary operations (=, +, -, *, /, %)
- comparisons (<, <=, >, >=, ==, !=)
- boolean operators (&&, ||)
- bitwise operators (&, |, ^)
- basic scope
- extern functions
- nested functions
- if/else statements
- for loops
- while loops
- loop control (continue, break)
- function control (return)
- modules can be defined
